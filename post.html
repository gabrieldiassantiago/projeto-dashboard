Como Fazer um Pedido de Venda

📁 Comercial → Movimentos → Pedidos

Resumo

Aprenda de forma rápida e prática como fazer vendas pelo sistema!

1 - Siga o Caminho

Logo abaixo, está o caminho para que você possa fazer o pedido de venda:

2 - Preencha os Dados

Após acessar a tela de pedidos, você precisará preencher os seguintes campos:

  1. Selecione o cliente na lista ou cadastre um novo
  2. Escolha a condição de pagamento
  3. Adicione os produtos desejados
  4. Confira os valores e descontos

3 - Finalize o Pedido

Para finalizar o pedido, siga os passos abaixo:

  1. Verifique se todos os dados estão corretos
  2. Clique no botão "Salvar" para registrar o pedido
  3. Imprima o pedido se necessário
